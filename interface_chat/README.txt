INSTRUCTIONS POUR UTILISER LES POLICES EN LOCAL

1. Déplacez les dossiers "fonts/roboto" et "fonts/material-symbols" dans votre projet.
2. Dans votre fichier HTML, ajoutez ceci dans la balise <head> :

<!-- Roboto -->
<link rel="stylesheet" href="fonts/roboto/roboto.css">

<!-- Material Symbols -->
<link rel="stylesheet" href="fonts/material-symbols/MaterialSymbolsOutlined.css">

3. N'oubliez pas d'utiliser les classes comme "material-symbols-outlined" dans votre HTML.

NOTE: Les fichiers WOFF2 ici sont vides. Remplace-les avec les vrais si nécessaire.
